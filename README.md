## ğŸš€ Getting Started

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/pjgs/olachea-group-website.git
   cd olachea-group-website
   ```

2. Start a local server:
   ```bash
   # Using Python
   python -m http.server 8000

   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```
